# Environment Variables Configuration

## Required Variables

Add these environment variables to your `.env.local` file:

```env
NEXT_PUBLIC_API_URL=https://backend-ai-learning-insight-production.up.railway.app
```

## Setup Instructions

1. Create a `.env.local` file in the root directory of your project
2. Copy the variables above into the file
3. Restart your development server

**Note:** The `.env.local` file is gitignored by default and should never be committed to version control.
